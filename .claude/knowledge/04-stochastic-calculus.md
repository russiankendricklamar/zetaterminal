# Stochastic Calculus — Formula Reference

Sources: Shiryaev (1998) Т.1 Гл.III, Т.2 Гл.VII-VIII;
Булинский, Ширяев (2005) "Теория случайных процессов";
Липцер, Ширяев (1974) "Статистика случайных процессов"

## 1. Wiener Process (Brownian Motion)

Определение (Ширяев I§3a, Булинский Гл.II-III):
W(t) — стандартный винеровский процесс если:
- W(0) = 0
- W(t) - W(s) ~ N(0, t-s) для t > s
- Приращения независимы: W(t₂)-W(t₁) ⊥ W(t₄)-W(t₃) для t₁<t₂≤t₃<t₄
- Траектории непрерывны п.н.

Properties:
- E[W(t)] = 0, Var[W(t)] = t
- Cov[W(s), W(t)] = min(s,t)
- Траектории нигде не дифференцируемы п.н. (Булинский Гл.III)
- Квадратическая вариация: [W,W]_t = t

## 2. Stochastic Integral (Ito)

Ширяев III§3c, Булинский Гл.IV:

Для предсказуемого процесса f(t):
```
I(t) = ∫₀ᵗ f(s) dW(s)
```

Свойства:
- E[I(t)] = 0 (мартингал)
- E[I(t)²] = E[∫₀ᵗ f(s)² ds]  (изометрия Ито)
- I(t) — непрерывный локальный мартингал

## 3. Формула Ито

Ширяев III§3d, Hull Appendix Ch.14:

Для f(t, X_t), где dX = μdt + σdW:
```
df = (∂f/∂t + μ·∂f/∂X + ½σ²·∂²f/∂X²) dt + σ·∂f/∂X · dW
```

Key insight: (dW)² = dt (в среднеквадратичном), поэтому второй порядок по dX не исчезает.

Многомерная версия (n процессов):
```
df = [∂f/∂t + Σᵢ μᵢ·∂f/∂Xᵢ + ½·Σᵢ Σⱼ σᵢσⱼρᵢⱼ·∂²f/∂Xᵢ∂Xⱼ] dt + Σᵢ σᵢ·∂f/∂Xᵢ · dWᵢ
```

## 4. Martingales in Finance

Ширяев Т.2, Гл.V-VI:

### Первая фундаментальная теорема
Рынок безарбитражный ⟺ ∃ эквивалентная мартингальная мера Q:
```
S̃(t) = S(t)/B(t)  — мартингал относительно Q
```
где B(t) = e^(rt) — банковский счёт (numéraire).

### Вторая фундаментальная теорема
Рынок полный ⟺ мартингальная мера Q единственна.

### Risk-Neutral Pricing (Ширяев VIII§1b)
Цена дериватива с выплатой f(S_T) в момент T:
```
V(0) = e^(-rT) · E_Q[f(S_T)]
```
Под Q: dS = rS·dt + σS·dW^Q (дрейф μ заменён на r).

## 5. Girsanov Theorem

Ширяев Т.2 V§3, Липцер-Ширяев Гл.6:

Замена меры P → Q с плотностью:
```
dQ/dP|_t = Z(t) = exp(-∫₀ᵗ θ(s) dW(s) - ½∫₀ᵗ θ(s)² ds)
```

Тогда W^Q(t) = W(t) + ∫₀ᵗ θ(s) ds — винеровский процесс под Q.

Для BSM: θ = (μ - r)/σ (рыночная цена риска / Sharpe ratio).

**Условие Новикова**: E[exp(½∫₀ᵀ θ(s)² ds)] < ∞ ⟹ Z(t) — мартингал.

## 6. Feynman-Kac Formula

Ширяев III§3f, Hull §15.6:

Если u(x,t) решает PDE:
```
∂u/∂t + μ(x)·∂u/∂x + ½σ²(x)·∂²u/∂x² - r·u = 0
u(x,T) = g(x)
```
то u(x,t) = e^(-r(T-t)) · E[g(X_T) | X_t = x], где dX = μdt + σdW.

Это связывает BSM PDE с risk-neutral pricing.

## 7. Stochastic Differential Equations (SDE)

Ширяев III§3e:

Общая форма: dX = μ(X,t)dt + σ(X,t)dW

### Geometric Brownian Motion
```
dS = μS dt + σS dW
S(t) = S(0)·exp((μ - σ²/2)t + σW(t))
```

### Ornstein-Uhlenbeck
```
dX = κ(θ - X)dt + σdW
X(t) = θ + (X(0) - θ)e^(-κt) + σ∫₀ᵗ e^(-κ(t-s)) dW(s)
```
E[X(t)] = θ + (X(0) - θ)e^(-κt), Var[X(t)] = σ²(1 - e^(-2κt))/(2κ)

### CIR (Square Root)
```
dX = κ(θ - X)dt + σ√X dW
```
X(t) ≥ 0 if 2κθ ≥ σ² (Feller condition)

## 8. Semimartingales

Ширяев III§5, Т.2 VII§1:

Семимартингал X = M + A, где M — локальный мартингал, A — процесс ограниченной вариации.

Разложение Дуба-Мейера (Ширяев III§5b):
Для субмартингала X: X = M + A, где A — предсказуемый возрастающий.

Квадратическая вариация:
```
[X,Y]_t = lim_{n→∞} Σᵢ (X(tᵢ₊₁) - X(tᵢ))(Y(tᵢ₊₁) - Y(tᵢ))
```

## 9. Lévy Processes

Ширяев III§1b:

Процесс с независимыми стационарными приращениями. Характеристика:
```
E[e^(iuX(t))] = exp(t·ψ(u))
ψ(u) = iγu - ½σ²u² + ∫(e^(iux) - 1 - iux·1_{|x|<1}) ν(dx)
```
где (γ, σ², ν) — тройка Леви-Хинчина.

### Примеры Леви-процессов для финансов:
- **Variance Gamma** (VG): ΔX = θΔG + σ√ΔG·Z, G — гамма-процесс
- **CGMY**: ν(dx) = C·e^(-G|x|)/(|x|^(1+Y))·1_{x>0} + C·e^(-M|x|)/(|x|^(1+Y))·1_{x<0}
- **Normal Inverse Gaussian** (NIG): X(t) = μt + βδ²I(t) + δW(I(t))

## 10. Filtering (Липцер-Ширяев)

### Kalman-Bucy Filter
Signal: dX = a·X dt + b dW₁
Observation: dY = c·X dt + d dW₂

Optimal estimate: X̂(t) = E[X(t) | Y(s), 0≤s≤t]
```
dX̂ = a·X̂ dt + (γ(t)·c/d²)(dY - c·X̂ dt)
```
where γ(t) = E[(X(t) - X̂(t))²] satisfies the Riccati equation:
```
dγ/dt = 2a·γ + b² - c²γ²/d²
```
